The code is an in-progress Matlab version of Multiscale Graph Correlation, 
which also implements some popular global correlation measures like distance correlation, hsic (kernel correlation), Mantel, Pearson, etc.

The R version is available on CRAN and https://github.com/neurodata/mgc.
The Python version is available on PyPi and https://github.com/neurodata/mgcpy.

The Python code is the reference implementation, both matlab and R are development.
